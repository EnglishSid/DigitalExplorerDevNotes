# DE Trends 2 CSV

## Required parameters


- Industry
- SubIndustry
- BusinessArea

## base API

`https://digitalexplorer.dxc.com/wwb/api/digitalExplorer/businessTrends?subIndustry=<VALUE>&businessArea=<VALUE>`

**If spaces use %20**
